# Astrology User Manual (Front-End)

> Module: `src/modules/astrology/*` (Astrology)

## 1) What this module does
Astrology provides horoscope and astrology-related content for users.

Depending on the current scope, it may include:
- Daily horoscope / zodiac-based insights
- Content by zodiac sign
- Basic astrology content browsing

## 2) Entry point in the app
1. Open the app dashboard.
2. Select **Astrology**.

## 3) Step-by-step user flow

### 3.1 View horoscope content
1. Open Astrology.
2. Browse the default horoscope view (often today’s content).
3. Read the horoscope details and recommendations.

### 3.2 Select your zodiac sign (if supported)
1. Find zodiac selection controls.
2. Choose your sign.
3. View the horoscope for that sign.

## 4) Troubleshooting (UI-level)
- Content not loading:
  - Refresh the page.
  - Check network connectivity.
  - Retry after reopening Astrology.

## 5) UI sections reference
- Horoscope/insights content area
- Zodiac sign selector (if supported)
- Tabs/filters for different content types (if present)
