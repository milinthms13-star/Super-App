# Quick Links User Manual (Front-End)

> Module: `src/modules/quicklinks/QuickLinks.js`

## 1) What this module does
Quick Links provides fast access to commonly used sections and actions. It typically serves as a shortcuts hub to reduce navigation time.

## 2) Entry point
1. Login.
2. Open **Quick Links** from app navigation (or dashboard).
3. The module shows a list/grid of shortcuts.

## 3) Step-by-step user flows

### 3.1 Open a shortcut
1. Locate a shortcut card/button on the Quick Links screen.
2. Click the shortcut.

Expected result:
- App navigates to the target module/screen.

### 3.2 Use multiple shortcuts
1. Repeat clicking different shortcut cards.
2. Return to Quick Links if navigation supports it.

Expected result:
- Each shortcut reliably navigates to the correct area.

## 4) Troubleshooting (UI-level)
- Shortcut doesn’t navigate:
  - Refresh the page.
  - Confirm your account/session is active.
  - Verify that the module is enabled for your role/subscription.

## 5) UI sections reference
- Shortcut grid/cards
- Shortcut actions/buttons
