# Canva Prompts: MalabarBazaar Classifieds Redesign

Use each prompt as a separate Canva Magic Design screen while keeping one consistent brand kit.

## Brand Kit
- Primary: `#0F766E`
- Primary Dark: `#0C4F56`
- Accent Gold: `#CF8F2D`
- Ink: `#132132`
- Muted: `#5F748A`
- Border: `#D4E4ED`
- Soft Surface: `#EFF6FB`
- White: `#FFFFFF`
- Danger: `#C03939`
- Fonts: Sora (headings), Manrope (body)

## Prompt 1: Classifieds Dashboard (Desktop)
Design a premium classifieds dashboard for MalabarBazaar with hero messaging, trust chips, KPI cards, and role mode cards (Buyer, Seller, Admin). Use rounded glassmorphism cards, layered gradients, and a trust-first marketplace tone. Include CTAs: Explore Listings, Post Ad, Manage Responses.

## Prompt 2: Filters + Listing Grid
Create a conversion-focused classifieds discovery section with advanced filters and listing cards. Include search, category, district, budget range, condition, verification status, and sort controls. Listing cards must show badges (verified, featured, urgent), title, price, location, seller info, trust indicators, and action buttons (Save, Chat, Report).

## Prompt 3: Listing Detail Panel
Design a premium listing detail screen with sections for overview, media, seller trust, map, quick actions, and chat. Keep a sticky action card with price, seller response-time badge, and buttons for chat/call/share. Add clear visual hierarchy and high readability.

## Prompt 4: Seller Posting Workspace
Design a seller-first ad posting UI with a structured form: title, category, pricing, location, description, media upload, tags, and language support. Include quality hints, spam checks, and monetization options (free, featured, urgent boost). Make the form feel guided and confidence-building.

## Prompt 5: Admin Moderation Console
Create a moderation dashboard for classifieds admins. Include pending approvals, spam queue, risk score cards, flagged keyword alerts, and bulk moderation actions. Use trust/compliance cues and clear severity states (safe, review, block).

## Prompt 6: Mobile Classifieds (390px)
Design a mobile-first classifieds experience with thumb-friendly controls, sticky quick actions, filter sheet, listing cards, and compact detail view. Keep all tap targets at least 44px and preserve premium visual quality with short, scannable content blocks.

## Prompt 7: Desktop Density (12-Column)
Design a high-density desktop classifieds layout using a strict 12-column grid. Set main marketplace content to 8 columns and sticky detail rail to 4 columns. Listing grid must also be 12 columns with standard cards spanning 4 columns and priority cards (featured or urgent) spanning 6 columns. Optimize spacing for fast scanning: compact badges, tighter metadata rows, reduced card padding, and short readable text lines while preserving premium look and trust cues.
