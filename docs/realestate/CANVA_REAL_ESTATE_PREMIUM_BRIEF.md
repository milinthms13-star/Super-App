# Canva Brief: Premium RealEstate Module

## Project Goal
Create a premium, trust-first, mobile-friendly Real Estate interface for "HomeSphere" with a modern Indian-market visual tone.

## Visual Direction
- Style: Premium professional, clean, trustworthy, conversion-focused
- Mood: Confident, transparent, high-value, fast-moving
- UI feel: Rich cards, soft gradients, high readability, clear actions

## Brand Tokens
- Primary: `#0F766E`
- Primary Dark: `#0C4F56`
- Accent Gold: `#CF8F2D`
- Ink: `#132132`
- Muted Text: `#60738A`
- Border: `#D7E3EE`
- Soft Surface: `#EFF6FB`
- White: `#FFFFFF`
- Danger: `#C03939`

## Typography
- Heading Font: Sora (Semibold/Bold)
- Body Font: Manrope or Poppins (Regular/Medium)
- Hero H1: 48-56 px desktop, 34-40 px mobile
- Section H2: 28-32 px desktop, 22-26 px mobile
- Body: 16-18 px

## Core Screens To Design
1. Marketplace Dashboard
2. Property Listing Grid
3. Property Detail (Tabbed)
4. Smart Filters Panel
5. Lead & Visit Management
6. Mobile View (full responsive adaptation)

## Layout Blueprint
- Desktop: 12-column grid, 24 px gutters, 80-120 px section spacing
- Tablet: 8-column grid
- Mobile: 4-column grid, 16 px horizontal padding
- Card radius: 20-24 px
- Button radius: 999 px (pill)
- Inputs: 14 px radius

## Key Components
- Premium hero with trust highlights
- Role-switch cards (Buyer/Owner/Agent/Builder/Admin)
- Smart filter bar with Apply + Clear + quick preset
- Property cards with:
  - trust badge
  - document score
  - possession status
  - media preview
- Detail tabs:
  - Overview, Media, Documents, Financing, Messages, Reviews, Actions
- Trust panel:
  - trust score
  - missing documents warning
  - verification CTAs

## Mobile Rules
- Keep primary actions thumb-reachable
- Minimum tap target: 44 px
- Stack filter/action controls vertically
- Reduce visual clutter; preserve trust data and pricing first

## Micro-interactions
- Card hover lift: subtle
- Button hover: slight elevation and tint
- Tab active state: high contrast pill
- Image hover: gentle zoom

## Copy Tone
- Crisp, practical, trusted marketplace language
- Avoid hype; emphasize verified data and action speed
- Focus on outcomes: shortlist, verify, contact, schedule

## Canva Build Sequence
1. Create Brand Kit with token colors and fonts
2. Build Desktop frame set
3. Duplicate to Tablet + Mobile and adapt layout
4. Convert repeated UI into reusable components
5. Export:
   - PNG screens for review
   - PDF product walkthrough
   - Optional prototype links

## Asset Checklist
- App logo mark
- Property photography placeholders
- Trust/status icon set (verified, pending, warning)
- Amenity icons
- Role icons

## Success Criteria
- Premium visual quality
- Fast scan readability
- Strong trust signaling
- Mobile-first usability
- Clear next actions in every section
