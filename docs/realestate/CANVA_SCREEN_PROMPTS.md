# Canva Prompts: HomeSphere Premium RealEstate UI

Use each prompt directly in Canva Magic Design (or Canva AI) as separate screens.  
Keep the same brand kit across all screens:
- Primary `#0F766E`
- Primary Dark `#0C4F56`
- Accent Gold `#CF8F2D`
- Ink `#132132`
- Muted `#60738A`
- Border `#D7E3EE`
- Soft Surface `#EFF6FB`
- White `#FFFFFF`
- Danger `#C03939`
- Fonts: Sora (headings), Manrope (body)

---

## Prompt 1: Marketplace Dashboard (Desktop)
Design a premium, trust-first real estate dashboard UI for a platform named HomeSphere. Desktop layout, 12-column grid, glassmorphism cards, soft gradients, rich but clean composition, Indian-market tone. Hero headline: "Find verified homes, rentals, and land in one place." Add two primary actions: "Explore Listings" and "Post Property". Show trust highlights chips: Verified Listings, Quick Connect, Multi-language Support. Include stats cards: Active Listings, Verified Listings, Rental Inventory, Open Leads, Document Checks Pending. Add role switch cards: Buyer/Tenant, Property Owner, Agent/Broker, Builder/Developer, Admin. Premium shadows, rounded 24px cards, excellent spacing, modern SaaS style, mobile-aware structure.

---

## Prompt 2: Smart Filter + Listing Grid (Desktop)
Create a premium real estate marketplace section with left filter controls and right/center property card grid. Include smart filters: search, listing intent, location, property type, max budget slider, min area slider, listing source, possession status, nearby signal, amenity, verified view, sort by. Include action buttons: Apply Filters, Clear Filters, Nearby + Verified. Show active filter count chip. Property cards must include: verification badge, featured badge, status badge, media preview image, title, price, location/locality, type/bed/area row, trust pill (High trust / Good trust / Trust pending), docs summary pill (x/6 docs), possession status, furnishing, leads count, rating, Save button. Make it premium, highly readable, and conversion-focused.

---

## Prompt 3: Property Detail (Tabbed, Desktop)
Design a premium property detail panel with top tab row: Overview, Media, Documents, Financing, Messages, Reviews, Actions. Include trust snapshot panel with trust status badge, trust score out of 100, missing documents warning line, CTA buttons: Request Documents and Verification History. In Overview tab include pricing, type, intent, specs chips, long description, amenities chips, map card, seller contact card, trust flags grid, and quick metadata row (listed date, leads, media assets). Keep visual hierarchy sharp, with compact but elegant card spacing and premium micro-contrast.

---

## Prompt 4: Documents & Verification Screen
Create a trust/document-focused UI section for property verification. Show verification items as cards with colored status dots: KYC, RERA, Title Deed, Tax Receipt, Building Permit, Encumbrance Certificate. Each row has state labels like Available/Pending/Rejected. Add a missing documents warning block and CTA buttons for requesting documents and viewing verification history. Use premium compliance style with high trust cues, subtle caution colors, and clean legal-tech aesthetics.

---

## Prompt 5: Lead & Visit Management (Desktop)
Design a CRM-style real estate operations area with two blocks: Lead Management and Visit Schedule. Lead cards show name, channel, priority, stage; include action button "Mark Contacted". Visit cards show date/time, mode (onsite/video), status chips, and quick status updates. Add a monetization block for subscription plans (Razorpay/Stripe) with featured credits and invoice summary. Style should look enterprise-premium and operationally clear.

---

## Prompt 6: Messaging + Reviews + Actions Tab
Design a tab content layout that includes:
- In-app messaging panel with chat history, input field, send button
- Reviews and reporting panel with rating dropdown, comment textarea, report textarea
- Action stack with buttons: Send Enquiry, Call Owner/Agent, WhatsApp Contact, Share Property, Download Brochure, Enable Alert
- Visit scheduler form with datetime input, visit mode select, notes, schedule button
Make this section compact, premium, and touch-friendly.

---

## Prompt 7: Mobile HomeSphere (Full Responsive)
Design a mobile-first premium real estate app screen set for HomeSphere (390px width). Keep actions thumb-reachable and minimum tap targets 44px. Stack cards vertically with rounded corners and clear spacing. Include: hero summary, filter accordion with apply/clear, property cards with trust/docs badges, tabbed detail switcher, trust panel, and sticky bottom action area for enquiry/visit. Preserve premium tone and readability with short content blocks and high-contrast buttons.

---

## Prompt 8: Tablet Adaptation
Create tablet adaptation (768px width) of HomeSphere premium real estate marketplace. Keep two-column rhythm where possible, convert dense desktop regions into stacked modular cards. Ensure trust panel, filters, listing grid, and detail tabs remain visually premium and easy to scan. Maintain brand palette and typography consistency.
