# Canva Prompts: MalabarBazaar Devadarshan Module

Use these prompts in Canva Magic Design as separate screens, while keeping one design system.

## Brand Kit
- Primary: `#0F766E`
- Primary Dark: `#0C4F56`
- Accent Gold: `#CF8F2D`
- Ink: `#132132`
- Muted: `#5F748A`
- Border: `#D4E4ED`
- Soft Surface: `#EFF6FB`
- White: `#FFFFFF`
- Danger: `#C03939`
- Fonts: Sora (headings), Manrope (body)

## Prompt 1: Devadarshan Dashboard (Desktop)
Design a premium Kerala devotional dashboard for a module named Devadarshan. Include hero message, quick action cards, temple discovery CTA, and trust chips. Keep rounded glass cards, layered gradients, and a calm devotional tone with modern product polish.

## Prompt 2: Temple Directory + Smart Filters
Create a conversion-friendly temple directory section with sticky filter row. Include search, district, deity, temple type, pooja type, festival, distance, and popularity filters. Temple cards should show verification badge, timings, pooja pricing hints, and quick actions: View Details, Save, Select.

## Prompt 3: Vazhipadu Booking Workspace
Design a two-column booking workflow for pooja booking with clear form sections: devotee profile, nakshatra, date, quantity, prasadam options, delivery mode, payment method, and notes. Add a side panel with selected temple details and expected amount summary.

## Prompt 4: My Devadarshan Tracking (Timeline First)
Design a booking lifecycle tracking screen with timeline cards and state chips: Pending, Confirmed, Completed, Cancelled. Include contextual CTAs: Pay Now, Track Timeline, Download Receipt, Cancel Booking. Add refund state visibility for cancelled paid bookings.

## Prompt 5: Donation + Receipt Vault
Create a donation page with category selection, amount input, purpose notes, and secure checkout CTA. Include a donation history table/list with receipt download actions and transaction references.

## Prompt 6: Live Darshan Experience
Design a live darshan section with embed-ready stream card, stream status chip, fallback message when stream unavailable, and actions: Open Live, Save Live, Notify Me. Keep layout touch-friendly and trust-oriented.

## Prompt 7: Admin Operations Console
Design an admin panel for temple verification, event publishing, and booking status updates. Show operational KPIs, booking queues, moderation actions, and audit-aware interaction patterns.

## Prompt 8: Mobile Devadarshan (390px)
Design a mobile-first Devadarshan app screen set with thumb-friendly controls and sticky bottom nav. Include quick cards, compact temple cards, simplified booking forms, and timeline cards with clear status chips.

## Prompt 9: Desktop Density (12-Column Ops View)
Design a high-density desktop Devadarshan view with strict 12-column rhythm. Use compact information cards for operations users: quick cards in 4x row pattern, temple/event cards in 3-column compact grid, and panels optimized for scanning long booking and donation lists. Keep typography readable but tighter, reduce card paddings, and preserve trust-first visual tone.
