# Master Documentation Index

This folder contains **master** documentation files that aggregate your existing per-module docs.

## Files
- `WORKFLOW_USER_MANUAL_MASTER.md` — End-to-end workflow (UI-level) manual across modules
- `FEATURES_IMPLEMENTED_MASTER.md` — Feature coverage implemented per module (roll-up)
- `TECHNICAL_DOCUMENTATION_MASTER.md` — Technical architecture roll-up per module

## How to use
- Start with `WORKFLOW_USER_MANUAL_MASTER.md` for “what users do”.
- Use `FEATURES_IMPLEMENTED_MASTER.md` to quickly review implemented capabilities.
- Use `TECHNICAL_DOCUMENTATION_MASTER.md` to understand structure, entry points, state models, and integration readiness.
